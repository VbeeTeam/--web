### 商品模块



| 字段          | 类型           | 备注                          |
| ------------- | -------------- | ----------------------------- |
| product_id    | Object(string) | 商品id 自动生成随机对象字符串 |
| product_name  | string         | 商品名字                      |
| product_img   | string         | 商品图片                      |
| product_price | number         | 商品价格                      |
| product_desc  | string         | 商品描述                      |
| product_cate  | number         | 商品分类 1：推荐 2：特价      |
